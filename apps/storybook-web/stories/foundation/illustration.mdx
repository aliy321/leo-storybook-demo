# Illustrations

Illustrations enhance the visual appeal of your application and make it more engaging for users.

Browse all available illustrations. Click an illustration to copy its name to the clipboard.

Use the **Components → Illustration** docs to display these illustrations in your application when the component ships in `@leo/ui`.

export const illustrationCount = Object.keys(illustrationSet).length;

export const getIllustrationSvg = (type, width = '140', height = '100') => {
  const svg = illustrationSet[type];
  if (!svg) return '';
  return svg.replace('<svg', `<svg width="${width}" height="${height}"`);
};

export const IllustrationGallery = () => {
  const allIllustrations = Object.keys(illustrationSet).sort();
  const [copiedIllustration, setCopiedIllustration] = useState(null);

  const handleIllustrationPress = async illustrationType => {
    try {
      if (typeof navigator !== 'undefined' && navigator.clipboard) {
        await navigator.clipboard.writeText(illustrationType);
      }
      setCopiedIllustration(illustrationType);
      setTimeout(() => setCopiedIllustration(null), 2000);
    } catch {
      setCopiedIllustration(illustrationType);
      setTimeout(() => setCopiedIllustration(null), 2000);
    }
  };

  return (
    <div style={{ background: 'var(--leo-color-canvas, #f7f7f7)', padding: 24 }}>
      <p style={{ margin: '0 0 8px', fontSize: 14, fontWeight: 600, color: 'var(--leo-color-foreground, #1a1a1a)' }}>
        {allIllustrations.length} illustrations available
      </p>
      <p style={{ margin: '0 0 24px', fontSize: 12, color: 'var(--leo-color-secondary, #666)' }}>
        Click any illustration to copy its name to the clipboard
      </p>
      <div
        style={{
          display: 'flex',
          flexWrap: 'wrap',
          gap: 16,
        }}
      >
        {allIllustrations.map(illustrationType => (
          <button
            key={illustrationType}
            type="button"
            onClick={() => handleIllustrationPress(illustrationType)}
            style={{
              width: 160,
              padding: 12,
              borderRadius: 8,
              border: '1px solid var(--leo-color-grey-300, #d4d4d4)',
              background:
                copiedIllustration === illustrationType
                  ? 'var(--leo-color-success-muted, #e8f5e9)'
                  : 'var(--leo-color-surface, #fff)',
              cursor: 'pointer',
              display: 'flex',
              flexDirection: 'column',
              alignItems: 'center',
            }}
          >
            <div
              style={{
                display: 'flex',
                alignItems: 'center',
                justifyContent: 'center',
                height: 120,
              }}
            >
              <span
                aria-label={illustrationType}
                role="img"
                style={{ display: 'inline-flex', width: 140, height: 100 }}
                dangerouslySetInnerHTML={{ __html: getIllustrationSvg(illustrationType) }}
              />
            </div>
            <span
              style={{
                marginTop: 8,
                fontSize: 10,
                textAlign: 'center',
                fontWeight: 600,
                color:
                  copiedIllustration === illustrationType
                    ? 'var(--leo-color-success, #2e7d32)'
                    : 'var(--leo-color-secondary, #666)',
              }}
            >
              {copiedIllustration === illustrationType ? '✓ Copied!' : illustrationType}
            </span>
          </button>
        ))}
      </div>
    </div>
  );
};

## Usage

```tsx
import { Illustration } from '@leo/ui';
import { View } from 'react-native';

// Basic usage (responsive)
<Illustration type="investment" />

// Custom dimensions
<Illustration type="wealth" width={300} height={200} />

// Fill parent width (percentage width preserves the SVG's intrinsic aspect ratio)
<View style={{ width: 320 }}>
  <Illustration type="wealth" width="100%" />
</View>

// With accessibility label
<Illustration type="success" accessibilityLabel="Success illustration" />
```

## Principles

- Keep illustration style consistent within a flow — do not mix illustration sets
- Respect brand palette and stroke styles defined by design
- Provide an `accessibilityLabel` when the illustration conveys meaning beyond decoration
- Do not use illustrations as the only way to complete a task

## When to use

| Moment | Guidance |
| ------ | -------- |
| Empty states | Reinforce why content is missing and what to do next |
| Onboarding | Introduce features with a single focal illustration |
| Success / confirmation | Celebrate completion without blocking next steps |
| Marketing modules | Support hero messaging — one illustration per module |
