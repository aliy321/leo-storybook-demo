# Card

Card groups related content in a contained surface.

**Platform:** React Native (`Card` from `@leo/ui/card` native entry)

**Supported screens:** Mobile, Tablet, Desktop

## Usage

- Use **elevated** for primary content blocks with subtle shadow.
- Use **outline** when you need a clear boundary without elevation.
- Use **filled** for lower-emphasis grouped content on the page background.
- Pass `onPress` to make the card pressable with Android ripple feedback.

## Accessibility

- Cards are presentational unless `onPress` is provided.
- When pressable, set `accessibilityLabel` if slotted content is not descriptive.
- Disabled cards must not trigger actions.
