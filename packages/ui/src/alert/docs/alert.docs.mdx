# Alert

Compound alert for contextual messages. Compose `leo-alert`, `leo-alert-title`, and `leo-alert-description` instead of passing title, body, and button labels as props.

**Supported screens:** Mobile, Tablet, Desktop

## Usage

- Use alerts near the content they describe.
- Keep titles short and descriptions specific.
- Add actions by composing `leo-button` inside the alert.

- Keep titles short and descriptions specific.
