# Alert

Compound alert for contextual messages. Compose `Alert`, `AlertTitle`, and `AlertDescription` instead of passing title, body, and button labels as props.

**Platform:** React Native (`Alert` from `@leo/ui/alert` native entry)

**Supported screens:** Mobile, Tablet, Desktop

## Usage

- Use alerts near the content they describe.
- Keep titles short and descriptions specific.
- Add actions by composing `Button` inside the alert.

- Root uses `accessibilityRole="alert"`.
- Keep titles short and descriptions specific.

Alert follows the shadcn compound pattern with shared CVA classes in `packages/ui/src/alert/core/alert.core.ts`.
