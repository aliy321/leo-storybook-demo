# Badge

Badge component scaffold — update this overview.

**Platform:** React Native (`Badge` from `@leo/ui/badge` native entry)

**Supported screens:** Mobile, Tablet, Desktop

## Usage

- Use **default** for the primary surface style.
- Use **outline** when you need a clear boundary without elevation.
- Pass `onPress` to make the component pressable.

## Accessibility

- Provide `accessibilityLabel` when content is not descriptive.
- Disabled state must not trigger actions.

## Implementation

Badge follows the CVA + semantic token pattern shared with Button and Card. Variants are defined once in `packages/ui/src/badge/core/badge.core.ts`.
